-Telar de jacquard 1801 
Es el primer sistema de almacenamiento de informacion binaria 0 y 1 o si o no 
el telar usaba un sistema de tarjetas perforadas para lograr diseños o dibujos complejos, si habia agujero la varilla pasada (podemos entenderlo como 1 o "encendido")
y si no habia agujero la varilla chocaba contra la tarjeta y quedaba abajo ( podemos entenderlo como 0 o "Apagado" 

El telar de Jacquard introdujo tres conceptos fundamentales de la computacion modernas

1. La separacion entre **Hardware*** y **Software** donde Hardware era la maquina fisica y Software serian las tarjetas perforadas antes tenias que reconstruir toda la maquina mientras con este sistema nada mas cambias de programa 
2. El sistema binario : creo un sistema de logica binario 
3. El pixel y la imagen rasterizada el tejido podemos entenderlo como una forma primigenia de la imagen de Mapa de bits
